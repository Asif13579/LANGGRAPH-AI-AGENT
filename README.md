# LangGraph AI Agent

An AI Agent built using LangGraph, LangChain, FastAPI, Streamlit, Gemini, and Tavily Search.

## Features

- AI Chat Interface
- Web Search Integration
- FastAPI Backend
- Streamlit Frontend
- Gemini LLM Support
- Tavily Search Tool

## Tech Stack

- Python
- LangGraph
- LangChain
- FastAPI
- Streamlit
- Gemini
- Tavily

## Run Locally

```bash
pip install -r requirements.txt
uvicorn backend:app --reload
streamlit run frontend.py
